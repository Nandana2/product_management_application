# Product Management Application

## Overview

This is a MERN Stack (MongoDB, Express.js, React.js, Node.js) application for managing products, categories, subcategories, and wishlists.

## Project Structure

## Getting Started

### Backend

```bash
cd Backend
npm install
npm start
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```
